# Security Policy

## Reporting a vulnerability

If you discover a security issue, please report it responsibly:

1. **Preferred:** Open a [GitHub private security advisory](https://github.com/mauricewipf/jugend-ui/security/advisories/new) on this repository.
2. **Alternative:** Open a GitHub Issue only if the report is not sensitive.

Please do not disclose security issues publicly before a fix is available.

## Scope

This policy covers the Jugend UI repository, demo site, and shadcn registry endpoints served from the project.

## Response

We aim to acknowledge reports within a reasonable timeframe and will work on fixes as capacity allows.
